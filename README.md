# Alpine Rice

Minimal Alpine Linux setup with a Parrot-style terminal.

---

## Install

Run on a fresh system:

```sh
apk add curl
curl -fsSL https://raw.githubusercontent.com/0xU53R/alpine-rice/main/install.sh | sh
```

---

## Reboot

```sh
reboot
```

---

## What it installs

* zsh + oh-my-zsh
* autosuggestions + syntax highlighting
* fastfetch (runs on terminal start)
* tmux (manual use)
* ssh enabled

---

## Prompt style

User:

```
┌─[user@hostname]─[~]
└──╼ $
```

Root:

```
┌─[ROOT@hostname]─[~]
└──╼ #
```
